# Face Recognition Attendance System

This is a simple project that uses Python and OpenCV to detect faces and mark attendance automatically.

## What it does
- Detects faces using webcam  
- Recognizes known faces  
- Saves name and time in a CSV file  

## Tech used
- Python  
- OpenCV  
- NumPy  

## How to run
1. Capture face images  
2. Train the model  
3. Run attendance script  

## Output
Attendance is saved in `attendance.csv`
